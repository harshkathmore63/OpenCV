# 🖼️ OpenCV Image Processing App

An interactive web application built using **Streamlit** and **OpenCV** that allows users to upload images and apply a wide range of image processing operations in real time.

---

---

## 📌 Features

* Upload images (JPG, JPEG, PNG)
* Real-time image processing
* Wide variety of OpenCV operations:

  * Grayscale conversion
  * Image resizing
  * Cropping with preview
  * Rotation & flipping
  * Gaussian & Median blur
  * Image sharpening
  * Edge detection (Canny)
  * Binary & adaptive thresholding
  * Drawing shapes (line, circle)
  * Add custom text on images
  * Face, eye & smile detection using Haar Cascades
* Side-by-side comparison (Original vs Processed)
* Download processed images instantly
